### BlissPermsAddon
ScoreHud addon for a pocketmine plugin [BlissPerms](https://github.com/XenoServers/BlissPerms).
### Available Tags
| Tag | Description |
|---------|---------|
| `{prefix}` | Show player prefix |
| `{suffix}` | Show player suffix |
| `{tier}` | Player Tier |
| `{rank}` | Player Rank |
| `{group}` | Player Group |
